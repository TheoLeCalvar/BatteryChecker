Battery Checker

Télécharger Code source sur GitHub

Battery Checker est un petit utilitaire pour MacBook qui affichera une notification quand votre batterie sera suffisamment rechargée pour vous avertir qu'il est temps de débrancher votre chargeur afin d'économiser de l'énergie !

Installation :

  1. Télécharger et ouvrir « BatteryChecker.dmg »

  2. Déplacer l'app « Battery Checker» dans le dossier « Applications »

  3. Double cliquer sur l'app (rien de visuel n'indiquera son lancement)

  4. Aller dans :

     > Préférences Système > Utilisateur et Groupes > Ouverture

    Cliquer sur le + pour ajouter l'application

  5. Aller dans :

     > Préférences Système > Notifications

    Selectionner « Battery Checker »

    Passer de « Bannière » à « Alerte »

    Décocher « Afficher dans Centre de notifications » et « Pastille sur l'icône de l'app »

  6. C'est fini ! L'application se lancera au démarrage et la notification restera visible pour être sûr de ne pas la louper !
